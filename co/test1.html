Exercice : QCM sur les systèmes experts

Quels sont les composantes d'un système expert ?

Votre choixChoix attenduRéponse

  • Un moteur d'inférence

  • Un réseau neuronal

  • Un base de fait

  • Une base de données

  • Une pierre d'infinité

  • Une base de règle

  • Une base Astrée

Comme dit précédemment, les systèmes experts possèdent 3 composantes : une base de fait, une base de règle et un moteur d'inférence. Le moteur d'inférence va puiser dans ces bases de faits et règles afin de créer de nouveaux faits jusqu'à pouvoir être en mesure de répondre à la question posée.
